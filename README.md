# MKRWAN-bme280
MKRWAN Lora BME280
Arduino MKR WAN 1300 or 1310  Shield for LoraWAN and The Things Network upload, with BME 280 pressure, humidity and temperature sensor.
Data uploaded to grafana dashboard on node red in a raspberry pi 3b+
